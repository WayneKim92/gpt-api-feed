# react-native-gpt-api-feed

Create feed to REST API using GPT

## Installation

```sh
npm install react-native-gpt-api-feed
```

## Usage

```js
import { multiply } from 'react-native-gpt-api-feed';

// ...

const result = await multiply(3, 7);
```
